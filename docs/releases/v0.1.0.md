# Agent Foundry v0.1.0 baseline

This prerelease freezes the first reproducible local MVP before Reliable Runs.

## Included

- TypeScript monorepo with Next.js, Fastify, worker, declarative workflows, versioned harness, artifacts, Git checkpoints, deterministic verification, model routing, and Codex/Claude/AGY adapters.
- Complete mock-mode integration path.
- Local filesystem persistence and directory queue.

## Supported environment

- Node.js 22+
- Git
- Trusted local operator
- `EXECUTOR_MODE=mock` for safe mechanics testing
- `EXECUTOR_MODE=real` only under the `real-local-trusted` deployment profile

## Known limitations

There is no durable `Run/StepRun/Attempt` recovery model, cooperative cancellation, queue lease, sandbox boundary, authentication, multi-tenancy, or hosted billing. Generated verifier scripts can run with host permissions. The API therefore binds to loopback by default for real execution.

## Validation

Run:

```bash
npm ci
npm run check
npm run doctor
```

The release target is the original public baseline commit recorded in `planning/governance-spec.json`; governance changes are delivered in a later commit.
